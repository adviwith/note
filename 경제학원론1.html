  • 경제학이란 무엇인가?

= 인간의 경제 행위를 연구하는 학문

인간의 욕구는 무한한데 비해 자원이 한정적이다. 이러한 희소한 자원의 배분(allocation)과 관련한 경제주체들의 선택행위를 경제 행위라고 한다.

경제 주체는 미시 경제학에서는 소비자, 기업 등이고 거시 경제학에서는 정부, 해외(수출입)등이다.

희소성(상대적인 개념)은 해소가 불가능하다. 인간의 욕구에 비해 자원은 한정적이기 때문 / 결핍, 빈곤(상대적이 아닌 절대적 빈곤)등은 이론적으로는 해소가 가능하다.

Think : 상대적 빈곤의 경우에는 인간의 욕망이 무한하기 때문에 생기는 것이므로 여기서 말하는 '선택'을 해야 하는 이유와 매우 유사한 배경을 가지고 있는 것 같다.

생산요소를 사용 → (생산과정) → 재화와 용역 → (소비과정) → 인간의 욕구 충족

  • 생산가능곡선(Production Possibilities Frontier(boundary, curve))

= 생산요소의 양이 주어진 상태에서 한 경제가 생산해낼 수 있는 최대 산출량들의 조합

= 생산해낼 수 있는 재화와 용역의 양과 그렇지 못한 양 간의 경계선, 도달가능(attainable) - 도달불가능(unattainable) 영역의 경계

양 극단의 점도 생산가능곡선상의 한 점이다.

M = 군수품 / C = 소비재

경계선상의 점에서는 효율적인 생산이 이루어진다. = 주어진 자원으로 M을 줄이지 않고는 C를 추가 생산하지 못하는 생태

경계선의 내부에서는 생산이 비효율적이다. = M을 줄이지 않고도 C 추가 생산 가능 = 자원 미활용/저활용

생산가능곡선상에서의 선택은 상반관계(배반관계, Trade-off)에 직면한다. 선택을 해야 한다.

M의 기회비용(o.c.) = 생산가능곡선의 기울기 / C의 기회비용(o.c.) = 생산가능곡선의 기울기

생산가능곡선이 밖으로 볼록한 이유 : M의 한계비용(MC)이 체증(증가)한다 = M생산을 증가시킴에 따라 M생산에는 덜 생산적인 자원(덜 생산적인 사람)들이 C로부터 M생산으로 전환되기 때문이다. - 그 역(M → C)도 성립

물고기와 토끼 두 가지를 사냥한다고 했을 때

단위 시간당

최대 A는 물고기 5마리, 토끼 1마리 / B는 물고기 2마리, 토끼 2마리 / C는 물고기 1마리, 토끼 6마리를 잡는다고 한다.

이 경우에 물고기나 토끼 사냥에 모든 사람을 투입할 경우 토끼 생산에는 비효율적인 A나 물고기 생산에는 비효율적인 B가 덜 생산적인 작업에 투입되므로 한계비용이 체증하는 것이다.

수요

(demand)

수요

수요 ≠ 수요량

수요량 : 구매자들이 특정 가격에 구입할 의사와 능력이 있는 수량 (실제로 그 가격이 되면 살 양)

원하는 수량 : 실제 구매되는 양은 아니다.(not actual quantity)

유효 수량 : 실제로 그 값이 되면 살 양 (the quantity buyers are willing to purchase)

"벤츠가 500만원이 되면 나도 사는거지, 돈도 있고."

유량(flow)

 

수요의 법칙(Law of demand)

= 한 재화의 가격이 상승하면 그 재화에 대한 수요량은 감소한다.

P↑ → Qd↓ = P↓ → Qd↑ : 수요곡선은 우하향한다.

= 대체효과와 소득효과가 작동하기 때문이다.

대체효과 : 가격이 오르면 소비자는 대체재를 구입해 수요량이 감소한다.

소득효과 : 가격이 오르면 소비자의 실질소득이 감소해 수요량이 감소한다. (정상재의 경우)

 

수요를 표시하는 3가지 방법 : 수요표, 수요곡선(demand curve), 수요(방정)식(demand equation)

블로그 참고

수요표 : 특정한 가격에 사람들이 살 양을 표(table)로 표시한 것

수요곡선 : 가격, 수요량

"곡선인데 쉽게 설명하기 위해서 직선으로 하는 거에요"

수요(방정)식 : Qd = 8 - 2P ( P = 4 - (1/2)Qd )

"종속변수, 독립변수 배웠겠...죠?"

독립변수 : 내가 어떻게 못하는 밖에서 주어진 '가격'

종속변수 : 내가 살 양

 

재화 가격의 변화 - 수요량의 변화 : 수요곡선 상의 점의 이동

관련 재화(대체,보완재)의 가격, 기대가격, 소득, 선호, 인구 등등 - 수요의 변화가 있으면 : 수요곡선 자체가 이동

 

  • 수요의 변화

수요의 증가 → 수요곡선 우로 이동

  1. 관련 재화의 가격

대체재(대신 쓰일 수 있는 재화) 가격 상승 → 수요 증가

보완재(함께 쓰일 때 효용성이 높아지는 재화) 가격 상승 → 수요 감소

  1. (미래의) 기대가격

미래에 가격 상승이 기대되면 현재의 수요 증가 → 수요곡선 우로 이동

후방굴절 수요곡선 : 주식, 부동산 등 주료 자산에 관해서 가격 상승이 기대되면 나타나기도 함

굴절수요곡선 // backward bending supply curve

  1. (소비자의) 소득

소득이 증가하면 (대부분의 경우) 재화를 더 소비한다 → 수요곡선 우로 이동

정상재(normal good) : 소득이 증가할 때 수요가 증가 eg. 대학교육 등

열등재(inferior good) : 소득이 증가할 때 수요가 감소 eg.감자, 라면

"우리나라에서는 경차는 이젠 열등재인데 인도에서는 그렇지 않아요."

"정상재, 열등재는 정해져있는 것이 아니라 소득수준에 따라 달라집니다."

자녀는 정상재일까요 열등재일까요? → 비인간적입니다! 빼애액 → 재(good) 얻기 위해 돈을 지불할 용의가 있는 것인데, 경제학의 기본이 안 되어 있는게 아닐까.. 재는 good, 좋은 것.

기펜재(Giffen good) : 열등재의 일종, 음의 소득효과가 대체효과를 압도해서 가격이 하락할 때 수요량이 감소한다. eg.감자가격감소 → 실질소득증가 → 고기사먹음 → 감자수요 감소

  1. (소비자의) 선호

선호의 변화에 따라 수요가 변화한다. eg.소고기, 담배, 운동기구

여기까지의 이야기는 우리가 재화의 퀄리티에 대한 정보를 가지고 있다는 전제 하에 이루어짐.

백화점에서 3만원짜리 물건을 100만원에 내놨을 때 사 가는 사람들이 있는 이유는 우리는 퀄리티에 대한 정보를 가격에서 받아들이기 때문이다. 가격을 질의 신호로 받아들임.

공급

(supply)

기업이 한 재화를 공급한다는 것은 그 기업이

[생산할 수 있는 기술과 자원을 갖고 있고, 생산해서 이윤을 얻을 수 있으며, 생산(및 판매)할 준비가 되어 있다는 뜻이다.]

공급량 : 특정 가격에 생산자들이 팔 용의가 있는 수량 : 유량

 

공급 법칙 : 가격이 상승하면 공급량은 증가한다.

= 생산에 있어서의 한계비용이 체증(increasing) : 생산자는 적어도 생산의 한계비용을 커버할 수 있어야 공급한다.

eg. 비용이 100인데 가격이 90면 공급하지 않는다.

가격이 상승하면 한계비용을 커버할 가능성이 높아지므로 보다 많은 공급자가 공급할 용의를 가진다.

 

  • 가격과 공급량간의 전체적인 관계 ↔ 수요

공급표 : 수요의 그것과 같음

공급 곡선 (우상향) : 최소공급가격(minimum supply price)곡선 = 특정 수량을 공급하게끔 하기 위해 제시되어야 하는 최소한의 가격

공급(방정)식 : Qs = -9 + 3P (← P = 3 + (1/3)Qs

공급방정식/수요방정식을 Q를 기준으로 쓰는 이유

= 가격은 개별 주체에게 주어지는 것이고, 임의로 변경하는 것이 아닌 독립변수이기 때문이다.

 

  • 공급량의 결정요소

그 재화의 가격 : 공급 곡선 상의 이동

그 외 요인의 변화 : 공급 곡선 자체의 이동 = 공급의 변화

 

  • 공급의 변화

공급증가 : 모든 가격에 대해서 공급량이 증가 → 공급곡선이 오른쪽(아래쪽)으로 이동한다.

  1. 재화 생상에 필요한 투입(생산요소=토지, 노동, 자본)의 가격이 증가

투입가격증가 → 생산비용증가 → 이윤감소 → 생산감소(생산포기) → 공급 감소 : 공급곡선이 왼쪽(위쪽)으로 이동한다.

  1. 기대가격(미래의 기대가격)

미래에 가격 하락이 기대되면 현재 공급 증가

eg. 아파트 (가격이 꼭짓점이다라고 판단하면 분양물량 증가)

주식, 부동산 모두 expectation game이에요

  1. 기술(기술진보)

기술진보 → 한계비용하락 → 이윤증가 → 공급증가 : 공급곡선이 오른쪾으로 이동한다.

 

시장 균형

(Market Equilibrium)

시장에서의 균형은 수요와 공급이 평형(balance)을 이룰 때 발생한다.

균형가격 : 수요량과 공급량을 일치시키는 가격

균형수량(균형 거래량, equilibrium volume) : 균형가격에서 사고 팔린 양

 

  • 시장의 조절자(regulator)로서의 가격 : 신회기능

: Qd와 Qs가 일치하지 않으면 일치할 때까지 가격이 변화한다.

시장이 청산(clead)된다 = 다 팔고, 모자라지도 않는다.

초과공급(잉여, surplus) → 가격에 하방 압력 → 가격 하락 (가격을 낮춰서라도 팜)

초과수요(부족) → 가격에 상방 압력 → 가격 상승 (돈을 더 주고라도 삼) eg.명절 기차표, 암표

"이러한 예시들은 농수산물 시장에서 자주 볼 수 있죠."

"잉여라는 단어는 나중에 또 나오니까, 초과공급으로 쓰는게 오해의 소지가 적습니다"

: 시장에서 균형으로 자동 복원력(automatic push)이 존재한다.

※단, 여기서의 균형은 정태적 균형(static equilibrium)이다.

정태적 = 시간에 따라 변하는 것을 고려하지 않는다 = 어떤 과정을 거쳐 균형에 도달했는지는 고려하지 않는다 = 가격이 올라갈 때 어떤 식으로 올라가는지는 고려하지 않는다 ↔ 동태적(dynamic equilibrium, eg.거미집 모형 등)

 

가치의 역설 : 사치재인 다이아몬드는 비싸고 필수재인 물값은 싸다 = 공급의 차이 : 그래프에서 공급곡선의 기울기, 위치가 다름

 

  • 비교정태(or 비교정학) 분석(Comparatie Statics)

정태적 균형(static equilibrium)을 비교하는 것

: 수요/공급의 변화 → 균형의 변화 : 두 개의 정태 균형을 비교한다.

  1. 수요 증가 → P증가 Q증가

대체제 가격의 증가, 소비자 소득 증가, 선호 변화 등으로 이루어짐

eg. 사과 시장

가격이 5일 때 공급 3으로 균형

→ 배(대체제)가 흉작인 경우

배 가격 상승 → 사과 수요가 2만큼 상승

수요 곡선 자체가 오른쪽으로 2만큼 이동함 - 새로운 균형 가격은 6

"이게 바로 정태 분석의 결과에요."

  1. 수요감소 → P감소 Q감소
  2. 공급증가 → P감소 Q증가
  3. 공급감소 → P증가 Q감소

탄력성

수요의 가격 탄력성

(Price Elasticity of Demand)

"그냥 수요탄력성(Demand Elasticity)이라고 불리기도 하는데,

실제로는 수요의 교차, 수요의 소득탄력성도 존재합니다."

가격이 상승할 때, 공급자들의 총수입(=소비자의 총지출)이 증가할까?

= 가격변화에 대한 수요 측 반응의 민감도의 문제 : 탄력성이라는 개념으로 정리하고 수치화 한다.

공급자들의 총수입(TR, Total Revenue : p × q) = 소비자의 총지출

수요의 가격탄력성 (e) = (－Qd의 퍼센트 변화/P의 퍼센트 변화 )

( '－'가 붙는 이유 = 가격하고 수요량의 변화는 반대이기 때문[음수 기울기를 가진 그래프])

최저임금이 10%오를 때 1%일자리가 감소하는건 숫자로 보면 별로 크지 않아 보이지만, 임금이 50%올랐을 때 일자리는 100만개가 줄어든다는 것을 의미.

 

가격이 오르면 수요량(Qd)는 감소한다 : 수요의 법칙

e = 탄력성, 여기에서는 수요의 가격 탄력성

e > 1 이면 수요는 탄력적이다.

( 수요량의 %감소 > 가격의 %증가) → 가격이 오를 때 총수입(TR)은 줄어든다

e > 1 일 때,

TR0 = p0 + q0

TR0 = ( p0 × 1.1 ) × ( q0 × 0.75 ) = 0.825 × p0q0

수요가 탄력적일 때 가격이 올라가면 총수입(=수요자의 총지출)은 줄어든다.

e < 1 이면 수요는 비탄력적이다

( 수요량의 %감소 < 가격의 %증가) → 가격이 오를 때 총수입(TR)은 증가한다

e = 1 이면 수요가 단위(unit)탄력적이다

→ 가격 변화 시 총수입(TR)의 변화는 없다.(총수입은 변하지 않는다.)

e = 1 일 때,

TR0 = p0 × q0

(1.01)p0 × (0.99)q0 = TR0 이므로 TR의 변화는 없다.

Think : 단위탄력적인건 뭔지 알겠는데 바로 위 식은 좀 틀린것처럼 보이기도..?

e = 0 이면 수요가 완전비탄력적(perfectly inelastic)이다

가격이 변화해도 수요량은 변하지 않는다 → 총수입 변화율 = 가격변화율

e = ∞ 이면 수요가 완전탄력적(perfectly elastic)이다

가격이 상승하면 수요량은 0으로 감소한다 → 총수입이 0으로

그래프가 거의 'ㅡ'모양으로 평탄할경우 가격이 조금 올라도 바로 수요가 0으로 수렴한다.

 

수요곡선이 직선이면 e는 좌상(왼쪽 위)에서 크고 우하(오른쪽 아래)에서 작다.

"탄력성은 모든 가격에서 똑같지 않습니다."

a = (10,1) / b = (9,2) / c = (2,10) / d = (1,11) 일 때,

a → b : p의 %변화 = -10%, q의 %변화 = 100% 이므로 [e = 10]

c → d : p의 %변화 = -50%, q의 %변화 = 10% 이므로 [e = 0.2]

= 똑같은 1감소, 1증가인데 e값이 다르다.

 

  • 탄력성의 수학적 표현

e = －( △Q/Q/△P/P ) = －(△Q/△P × P/Q) = －( 수요량의 %변화율/가격의 %변화율 )

※ －(△Q/△P)를 사용하지 않는 이유 : 변화'량'을 이용하면 무슨 단위를 쓰느냐에 따라 값이 멋대로 변하므로 어떤 단위에서든 동일한 변화'율'을 사용한다.

 

  • 호탄력성(arc elasticity)

평균가격과 평균수량(두 점의 중간점)을 이용해서 탄력성을 계산한다.

" 수요곡선을 알면 탄력성을 알 수 있지만, 현실에서는 몇 개의 점만 관찰 가능합니다."

a = (10,8) / b = (6,12)일 때

a → b 의 e는 5/4

b → a 의 e는 1/2

: 값이 다르다. (그래프에서 왼쪽으로 갈수록 탄력성 값이 크기 때문이다)

= 왼쪽으로 갈수록 가격변화의%가 작아지고 수요변화의%값이 커지기 때문이다.

그러므로 중간(평균)에서 잰다. = 호탄력성

( 평균가격, 평균수량 ) = ( 6+10/2 , 8+12/2 )

e(호탄력성) = －( 12-8/6-10 ) × 8/10 = 4/5

e(호탄력성) a → b = e(호탄력성) b → a = 같다

 

  • 점탄력성(point elasticity)

수요곡선의 기울기를 이용해 탄력성을 계산한다.

e = －( dQ/dP × P/Q )

= －( 1/수요곡선의 기울기 × P/Q )

 

 

  • 수요곡선과 총수입(TR)

수요곡선에서 e = 1이 되는 Q값에서 총수입은 극대화된다(기울기 = 0)

수요의 가격탄력성 곡선을 그리면 Q=0으로 갈수록 e=∞에 수렴하고 e=0으로 갈수록 Q=∞에 수렴하는 그래프가 그려진다. (양 끝에서 각 완전탄력적이고 완전비탄력적이다, x축-Q y축-e)

수요가 완전탄력적인 경우 수요곡선은 'ㅡ'모양으로 그려진다.

단위탄력적인 수요곡선은 ( eg. p = 10/q ) 와 같은 함수의 그래프이다. ( p×q = 10 = TR, 어느 점에서나 총수입이 상수로 일정하기 때문이다.)
